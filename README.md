# React 组件库 Buffalo Design

## 安装

```
npm install buffalo-design
```

## 使用

### ESM

```
import { Popover } from 'buffalo-design';
```

### CJS

```
const { Popover } = require('buffalo-design');
```

### UMD

```
<script src="https://unpkg.com/buffalo-design@0.0.2/dist/umd/buffalo-design.js" />
```
